## 🚀 Projetos

### [Thaifit Marmitas](https://github.com/DouglasHurtado/thaifit-marmitas) — [thaifitmarmitas.com.br](https://thaifitmarmitas.com.br)
Site completo de marmitas fitness com montador interativo, painel administrativo e pedidos via WhatsApp.

- Interface responsiva mobile-first em HTML, CSS e JavaScript puro
- Backend em Python com autenticação segura e API de preços
- Deploy em VPS próprio com nginx, HTTPS e fail2ban
- Primeiro projeto web do zero até a produção
